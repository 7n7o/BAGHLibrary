## Info
This is a library to interface with a vulnerable version of the pompous the cloud gear found in some roblox games.

## Building
To build to an RBXM run build.sh
To build to a single file run bundle.sh

### Scripts
To build a script bundle, run scripts/bundle.sh <path/to/script> <path/to/bundle> <(dev|prod)?>
Example: `scripts/bundle.sh scripts/lua/test.lua scripts/bundle.lua dev`
